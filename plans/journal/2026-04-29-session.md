## Key Decisions

1. **Tutorial uses plain Java, not Spring Boot** — AgentClient has static factories that work standalone
2. **LOOSE derives `dangerouslyBypassSandbox=true` for Codex** — matches terminal-bench's approach; bwrap sandbox fails in too many environments
3. **Property prefix `agent-client.*`** — distances from Spring AI namespace per their team's feedback
4. **`AgentApi` over `AgentModel`** — wraps CLI runtimes, not AI models
5. **Codex default model `gpt-5.4-mini`** — old `gpt-5-codex` removed in CLI 0.125.0
